## 🧪 Physics Background

The Quantum Wave Explorer visualizes solutions to the time-independent Schrödinger equation:

```
-ℏ²/2m ∂²ψ/∂x² + V(x)ψ = Eψ
```

Key concepts visualized:
- Wave functions (ψ)
- Probability densities (|ψ|²)
- Energy levels
- Quantum tunneling
- Node patterns
